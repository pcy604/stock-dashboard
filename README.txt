  주식 스크리너 사용법
═══════════════════════════════════

【실행 방법】
  run.bat 더블클릭 (또는 터미널에서 python screener.py)

【설정 변경】
  config.py 파일에서 변경:
  - USE_US_MARKET / USE_KR_MARKET : 시장 on/off
  - US_UNIVERSE : "SP500" / "NASDAQ100" / "CUSTOM"
  - KR_UNIVERSE : "KOSPI200" / "KOSDAQ150" / "CUSTOM"
  - MIN_SIGNALS_TO_SHOW : 신호 몇 개 이상일 때 출력할지 (기본: 2)

【감지하는 신호 6가지】
  1. 52주신고가  : 52주 최고가 3% 이내 또는 돌파
  2. 5일선라이딩 : 5일선 위에서 5일+ 연속 상승
  3. 컵위드핸들  : 컵 형성 후 돌파 12% 이내
  4. 이평선수렴  : 골든크로스 또는 이평선 수렴 중
  5. RSI/MACD   : MACD 골든크로스 / RSI 과매도 탈출
  6. 거래량폭발  : 평균 2.5배+ 거래량 + 강한 양봉

【결과 파일】
  results/ 폴더에 날짜별 CSV 자동 저장

【처음 실행 시 시간】
  - SP500 전체: 약 15~30분
  - 커스텀 (10종목): 약 2분
